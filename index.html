Welcome to nisbus@github.

This is the place to list some of the repositories I have created on github. I love to write Erlang code and try to write a few OS projects each year for fun. I hope you find something here useful (or funny :) ).

Erlang Repositories

  • erlang-measurements An application to convert measurement, supports currencies, metric/imperial and temperatures.
    Build Status
  • erlang-dates An application to create date lists for a specified range.
    [![Build Status](https://travis-ci.org/nisbus/erlang-dates.png?branch=master)](https://travis-ci.org/nisbus/erlang-dates)
  • Observerl Observer for streams of data that supports filtering, aggregating and windowing/buffering.
    [![Build Status](https://travis-ci.org/nisbus/Observerl.png?branch=master)](https://travis-ci.org/nisbus/Observerl)
  • winderl A more advanced windowing mechanism for streams of data than the one included in Observerl. [![Build Status](https://travis-ci.org/nisbus/winderl.png?branch=master)](https://travis-ci.org/nisbus/winderl)
  • babelstats A data warehouse for numerical data.
    [![Build Status](https://travis-ci.org/nisbus/babelstats.png?branch=master)](https://travis-ci.org/nisbus/babelstats)
  • monterl_carlo A Monte Carlo simulation. [![Build Status](https://travis-ci.org/nisbus/monterl_carlo.png?branch=master)](https://travis-ci.org/nisbus/monterl_carlo)
  • monterl_carlo_websocket A demo for the monterl_carlo repo, charts the data from the simulation on a web page using websockets and flot.
    [![Build Status](https://travis-ci.org/nisbus/monterl_carlo_websocket.png?branch=master)](https://travis-ci.org/nisbus/monterl_carlo_websocket)
  • erlang-stomp-client A simple stomp client.
    [![Build Status](https://travis-ci.org/nisbus/erlang-stomp-client.png?branch=master)](https://travis-ci.org/nisbus/erlang-stomp-client)
  • CouchJuke An expirement in creating a music library manager/player in couchdb.
  • couchdb_odata A simple show and a list for couchdb to serve your data as an odata feed.

.NET Repositories

  • propeller.framer A framework for communicating with the Parallax Propeller chip in .NET
