Welcome to nisbus@github.

This is the place to list some of the repositories I have created on github. I love to write Erlang code and try to write a few OS projects each year for fun. I hope you find something here useful (or funny :) ).

Erlang Repositories

  • erlang-measurements
    An application to convert measurement, supports currencies, metric/imperial and temperatures.

  • erlang-dates An application to create date lists for a specified range.

  • Observerl
    Observer for streams of data that supports filtering, aggregating and windowing/buffering.
  • winderl
    A more advanced windowing mechanism for streams of data than the one included in Observerl.
  • babelstats
    A data warehouse for numerical data.
  • monterl_carlo
    A Monte Carlo simulation.
  • monterl_carlo_websocket
    A demo for the monterl_carlo repo, charts the data from the simulation on a web page using websockets and flot.
  • erlang-stomp-client
    A simple stomp client.
  • CouchJuke
    An expirement in creating a music library manager/player in couchdb.
  • couchdb_odata A simple show and a list for couchdb to serve your data as an odata feed.

.NET Repositories

  • propeller.framer A framework for communicating with the Parallax Propeller chip in .NET
